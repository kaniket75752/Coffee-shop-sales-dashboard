# Coffee-shop-sales-dashboard
An interactive Excel dashboard built using Pivot Tables, Pivot Charts, KPI Cards, and Slicers to analyze revenue, profit, sales trends, product performance, and city-wise performance. The dashboard provides actionable business insights through dynamic visualizations and data-driven reporting.

## Dataset
- <a href="https://github.com/kaniket75752/Coffee-shop-sales-dashboard/blob/main/Coffee%20Shop%20Sales%20Data%20(1)%20(1).xlsx"></a> Dataset
- <a href="https://github.com/kaniket75752/Coffee-shop-sales-dashboard/blob/main/Screenshot%202026-06-09%20231649.png"></a> Dashboard

Key Insights:
Chicago generated the highest revenue among all cities.
Desserts were the top-performing product category in terms of revenue.
Total Revenue: ₹46.37K
Total Profit: ₹16.23K
14,117 units sold across 9,971 transactions.
Sales remained relatively consistent throughout the analysis period.
Top-performing products contributed significantly to overall revenue growth.


Benefits of the Dashboard:
Provides a quick overview of business performance.
Helps identify top-performing cities and product categories.
Tracks revenue, profit, and sales trends in real time.
Supports data-driven decision-making.
Enables interactive analysis using slicers and filters.
Helps businesses identify growth opportunities and improvement areas.
Saves time by presenting complex data in a visual and easy-to-understand format.
